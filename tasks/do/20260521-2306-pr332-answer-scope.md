# PR332 回答送信スコープ修正

状態: do
タスク種別: 修正

## 背景

PR #332 の再レビューで、`POST /questions/{questionId}/answer` が `answer:publish` を持つ通常 `ANSWER_EDITOR` に対して担当外 ticket の回答送信を許していると指摘された。

## 受け入れ条件

- 担当外 `ANSWER_EDITOR` は `POST /questions/{questionId}/answer` で 404 を受ける。
- 担当外失敗 response に `title`, `question`, `internalMemo`, `sanitizedDiagnostics` を含めない。
- `assigneeUserId === user.userId` の `ANSWER_EDITOR` は回答送信できる。
- `assigneeGroupId` が `supportGroupIds(user)` に含まれる `ANSWER_EDITOR` は回答送信できる。
- `support:ticket:read:all` を持つ管理者は担当外 ticket でも回答送信できる。
- 関連 API test / lint / typecheck を通す。
