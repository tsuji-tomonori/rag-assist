状態: doing

## 背景

Wave 3-pre の E-pre-gap として、仕様 3A/3C 章と既存 chunker / text extract / ingest worker 実装の差分を調査する。
後続 `E-parsing-and-chunkers` が実装すべき範囲と、既存の citation / drawing / page gate など踏襲すべき制約を明確にする。

## 目的

- 後続 `E-parsing-and-chunkers` の scope / scope-out と preserve 条件を明示する。
