# PR332 再レビュー指摘対応

状態: done
タスク種別: 修正

## 背景

PR #332 の再レビューで、`mergeable=false`、default support group の CDK 本番 env 未設定、default group と role filter の曖昧さ、履歴 favorite enrichment 前 limit、Favorite 再保存時の既存値上書きが指摘された。

## 受け入れ条件

- PR branch が `origin/main` を取り込み、競合解消済みになる。
- CDK template の API / Heavy API / worker 環境変数に `DEFAULT_SUPPORT_ASSIGNEE_GROUP_ID` が設定される。
- default support group が application role 名と一致する場合でも、担当者一覧の group query から除外されない。
- CHAT_USER が assignee 未指定で作成した ticket を、SYSTEM_ADMIN ではない通常担当者が一覧で取得できる。
- ConversationHistory の favorite enrichment 後に favorite 優先 sort と最終 limit を適用する。
- Favorite 再保存時に既存 `createdAt` と未指定 `note` / `label` を保持し、`updatedAt` と明示指定 field だけ更新する。
