## 1. 受けた指示

- チャット欄送信のキーバインドを切り替え可能にする。
- `Ctrl+Enterで送信 / Enterで改行` と `Enterで送信 / Shift+Enterで改行` のモード切替を実装する。

## 3. 検討・判断したこと

- 既存挙動が Enter 送信だったため、デフォルトは互換性重視で `Enterで送信` を維持した。
- 送信キー設定はチャット入力近くに配置し、利用者が文脈内で切り替えられるようにした。
- キー判定は `onKeyDown` に集約し、モードごとの分岐を明確にした。

## 4. 実施した作業

- `App.tsx` に送信キーモード state を追加。
- textarea の placeholder と Enter キー処理をモード連動で変更。
- 送信キーモード切替用の select を composer 下部に追加。
- `App.test.tsx` に Ctrl+Enter モードの送信・改行挙動テストを追加。
- 依存未導入環境だったため `pnpm install` 後に `pnpm test` を実行して回帰確認。
