## 3. 検討・判断したこと

- P1 は LLM rewrite ではなく、既存 deterministic rewrite の検索反映を本格化する方針にした。外部 model 挙動に依存せず、既存 unit test で検証しやすいため。
- clue 生成で `expandedQueries` が上書きされていたため、standalone query / retrieval query 群を保持して evidence search に渡すようにした。
- previous citation prior は answer evidence ではなく rerank score の soft boost に限定した。履歴回答だけを根拠にしない制約を守るため。
- `Citation` に optional `pageStart` / `pageEnd` を追加した。page-aware prior と debug / benchmark の根拠確認を可能にするため。
- P2 は agent rerank と search `cheapRerank` の両方に lightweight feature を追加した。cross-encoder は provider 設計が別 task になるため今回は入れていない。
