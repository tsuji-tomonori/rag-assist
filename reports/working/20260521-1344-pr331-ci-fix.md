## 3. 検討・判断したこと

- web lint は、右ペイン撤去後に表示先がなくなった state/handler と、未使用 props が残っていたことが原因だったため、到達不能コードを削除した。
- Web coverage は、廃止済み右ペインの `DocumentDetailPanel` とその utility が coverage 対象に残っていたこと、文書共有/移動 hook のテストが不足していたことが原因だったため、削除済み UI の実装ファイルを除去し、現行 UI/API hook のテストを追加した。
- OpenAPI docs check は、新規 route の日本語 operationDocs と生成 Markdown が不足していたため、runtime source に docs を追加して生成物を同期した。
- Web inventory check は生成物 stale だったため、生成コマンドで同期した。

## 4. 実施した作業

- `apps/api/src/openapi-doc-quality.ts` に文書共有/文書移動 route の日本語 summary/description を追加した。
- `docs/generated/openapi*` を再生成した。
- `docs/generated/web-*` を再生成した。
- `apps/web/src/features/documents/components/workspace/DocumentDetailPanel.tsx` を削除した。
- `DocumentWorkspace.tsx` から右ペイン専用の最近操作・新規フォルダ作成残コードを削除した。
- `DocumentFilePanel.tsx` の未使用 props を削除した。
- `documentWorkspaceUtils.ts` から未使用になった右ペイン専用 utility を削除した。
- `documentWorkspaceUtils.test.ts` と `useDocuments.test.ts` に coverage gate 用ではなく現行挙動の回帰テストを追加した。
