## 3. 検討・判断したこと
- 問題の本質は `workflow_dispatch` で任意ブランチ実行できる点と、checkout が信頼済み ref に固定されていない点と判断。
- 既存機能を維持するため、ワークフロー全体は維持しつつ `main` 限定のジョブガードと checkout ref 固定のみ追加する方針を採用。

## 4. 実施した作業
- `.github/workflows/memorag-deploy.yml` を確認し、問題のパターンが現存することを確認。
- `deploy` ジョブに `if: ${{ github.ref == 'refs/heads/main' }}` を追加。
- `actions/checkout@v4` に `with.ref: refs/heads/main` を追加。
- `git diff --check` でフォーマット異常がないことを確認。
