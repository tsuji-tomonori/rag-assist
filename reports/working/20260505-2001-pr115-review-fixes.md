## 3. 検討・判断したこと

- `llm_judge_invocation_rate` と `llm_judge_resolved_rate` は row 数を分母にするため、`rows_with_llm_judge_calls` の note は row denominator として残した。
- `llm_judge_no_conflict_rate` / `llm_judge_conflict_rate` / `llm_judge_unclear_rate` は call 数が分母なので、`llm_judge_call_count` を Coverage に追加した。
- refusal は `refusal_precision` と同じ `row.evaluation.refused` predicate に合わせ、Coverage item も `refused_rows` に改名した。
- 中期的な metadata 集約案は妥当だが、今回の PR ではレビュー指摘2点に閉じて小さく修正した。
